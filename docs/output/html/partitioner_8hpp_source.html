11// Central utility for domain decomposition in the row-wise (block-row) sense.
12// It knows:
13// - n_global : total number of unknowns
14// - rank, size : MPI info
15// - ls, le : local range [ls, le) of global row indices
16// - rowStarts : full partition [0..n_global] split among ranks
17//
18// It also provides helpers to:
19// - build the local matrix A_loc and RHS b_loc from a global A,b
20// (in the first version, assuming A,b exist on each rank)
21// - extract local slices of any global vector
22// - gather a distributed vector back to root.
